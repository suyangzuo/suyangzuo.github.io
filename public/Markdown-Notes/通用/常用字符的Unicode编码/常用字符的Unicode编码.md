## 标点符号

- **°** : `U+00b0` ⮕ 角度
- **•** : `U+2022` ⮕ 项目符号
- **·** : `U+b7` ⮕ 分隔点
- **×** : `U+00d7` ⮕ 标准乘号
- **✕** : `U+2715` ⮕ 错误
- **✓** : `U+2713`
- **⋯** : `U+22ef`

## 自然科学

- **π** : `U+03c0`
- **℃** : `U+2103`
