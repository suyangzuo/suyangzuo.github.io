格式说明符原创

作者： 苏扬

日期： 2024年2月15日

定义与用途

定义

格式说明符，是输入和输出函数中用于表示数据类型的占位符。

用途

格式说明符用于确保程序根据指定的数据类型正确读取和显示数据。当使用格式化输入或输出函数如：printf、scanf时，它们告诉编译器如何解释和显示数据。

格式与类型

格式

在C语言中，格式说明符由特殊字符表示，这些字符以百分比符号%开头，后跟一个表示数据类型的字符。每个格式说明符对应一个特定的数据类型，确保程序正确处理数据。

类型表格

格式说明符
格式说明符 含义 对应数据类型
%d 有符号整数 int
%u 无符号整数 unsigned int
%f 单精度浮点数 float
%lf 双精度浮点数 double
%c 字符 char
%s 字符串 char[] / char*
%p 指针 void*
%zu 对象的最大占用空间 size_t
%x 16进制无符号整数 unsigned int
%% 百分号 %
%hd 短有符号整数 short int
%hu 短无符号整数 unsigned short int
%ld 长有符号整数 long int
%lu 长无符号整数 unsigned long int
%lld 超长有符号整数 long long int
%llu 超长无符号整数 unsigned long long int
%Lf 长精度浮点数 long double
%llx 超长16进制无符号整数 unsigned long long int
%e 科学记数法形式的浮点数 double
%zd 可能的负空间占用或偏移 ssize_t
