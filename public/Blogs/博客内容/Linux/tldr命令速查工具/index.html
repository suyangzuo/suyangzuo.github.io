命令速查工具

man

man是Linux的原生命令速查工具，它的缺点比较明显，就是信息太多，容易吓到初学者。

tldr

tldr是什么意思？

tldr是Too Long; Didn't Read的英文首字母拼写，即太长，未读的意思。

tldr-pages是一个由社区维护的命令行速查工具集合，旨在成为man的一个更简洁易用的替代。

tldr用法

下载tldr

官方仓库页面推荐使用npm进行安装。

      
        npm install -g tldr

安装完毕后，使用tldr --version进行确认：

使用tldr

使用tldr 命令的方式，查询命令用法，如：

      
        tldr lsof

运行命令后，会得到一份清晰、易读的命令速查表。

更新tldr

tldr的本质就是一份份由贡献者提供的Markdown文档，每次运行命令，就会调用对应的文档并打印到终端。这些文档托管在GitHub上，同样会时不时更新。因此，我们每过一段时间就要更新相关文档的本地缓存，确保内容是最新的： 

      
        tldr --update

运行命令后，会下载最新文档，并重建本地搜索索引。
