浏览器引擎

浏览器引擎概述

  • WebKit：苹果公司开发，用于Safari以及以及其它苹果系统上的浏览器。
  • Blink：谷歌公司开发，从WebKit分支而来，用于Chrome、Edge等浏览器。

    小知识

    苹果公司开发了WebKit引擎，谷歌最早也是使用这个引擎开发浏览器的。后来，谷歌根据自身需求，从WebKit分支出了Blink引擎，在此基础上加上V8 JavaScript 引擎，创建了Chromium项目。Chromium是一个免费开源的浏览器项目但不是浏览器，任何人都可以用这个项目来开发浏览器。

    基于Chromium项目的浏览器，现在已经占据了绝大部分市场；最有名的就是Chrome和Edge。由于其核心的复杂历史，这类浏览器有很多称呼：

    • WebKit 核心其实它应该特指苹果公司开发的WebKit核心的浏览器，主要用于苹果设备
    • Blink 核心
    • Chromium 核心

    以上说法，其实都是同一个意思，只不过不同的人从不同的角度称呼它们而已。

  • Gecko：Mozilla公司开发，用于Firefox浏览器。

小知识

有人的地方就有江湖，浏览器行业亦不例外。每家公司都希望自己的浏览器能够垄断市场。比如苹果，它规定在所有的苹果系统设备上Mac、iPhone、iPad ……只能使用WebKit核心的浏览器。然而，WebKit核心就是苹果开发的，谁又能用得比它更好呢？结果就是在苹果系统设备上运行的基本都是苹果自家的Safari浏览器，形成了事实上的垄断。

2024年1月25日，苹果公司公布了针对欧盟新数字法案进行修订的iOS、Safari和应用商店新规则，从iOS 17.4开始，苹果不会再强制欧洲用户使用WebKit核心的浏览器，新规则于2024年3月在欧洲的27个国家生效。然而，很多开发者，包括开发了火狐FireFox浏览器的Mozilla公司却对此表达了巨大的不满和失望。

苹果的新规则仅仅开放了iOS上的浏览器限制，并且仅针对欧洲用户。很多开发者都觉得新规则非常可疑。

总而言之，浏览器绝不只是一个逛网站的工具这么简单，它已经成为互联网时代最重要的交互门户，赢得浏览器市场的重要性不亚于数十年前赢得操作系统市场。

提醒

自2025年7月27日起，火狐浏览器不再有国内版，具体信息可参考关于 Firefox 在中国持续运营但北京火狐关闭及 Firefox 账户服务终止的重要公告：

这也意味着持续了15年之久的火狐浏览器国内版、国际版之分正式宣告结束，因此本提醒的后续内容已经过时，可以忽略。

Firefox是这几个浏览器中唯一一个在中国大陆有代理的，因此也分成了国际版和国内版很多人都不知道有这回事。但凡电脑软件用的多、用的久，有经验的都知道，只要分了国际版和国内版，那国内版定然没有好事，可能出现的情况有：

  • 国际账号和国内账号不互通
  • 国内无法使用部分软件功能或服务如Adobe系列
  • 国内版无法获取特定特性，比如防广告扩展如Firefox

Firefox在国内由北京谋智公司代理，打开火狐浏览器的帮助菜单，如果有北京谋智，那就是国内版。

关于信息中包含了北京谋智，这就是国内版。

关于信息中只有Mozilla，这就是国际版。

我们肯定要使用国际版，但很多人往往在不知情的情况下下载了国内版，如何区分它们？

  • 国际版：https://www.mozilla.org/zh-CN/firefox/，域名是mozilla.org
  • 国内版：https://www.firefox.com.cn/，域名是firefox.com.cn

特别提醒：如果直接访问国际版域名，会被重定向到国内版域名！

直接访问mozilla.org，结果被自动重定向到firefox.com.cn，很多人都被这瞒天过海的招数骗了。因此，要下载国际版，需要直接访问国际版的Firefox 页面，而不是主页。
