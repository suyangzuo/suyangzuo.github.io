Windows 官方终端简介

Windows 官方终端有哪些

Windows的官方终端目前有2个：

  • ①Windows 控制台主机传统，也就是我们常说的 命令提示符cmd
  • ②Windows 终端现代

什么是命令提示符

命令提示符是大多数Windows操作系统中可用的命令行解释器应用程序，用于执行输入的命令。

提醒

命令提示符有时被错误地称为“DOS提示符” 或 MS-DOS。命令提示符是一个Windows程序，它模拟MS-DOS中的许多命令行功能，但它不是MS-DOS。

命令提示符有什么缺点

  • 界面丑陋
  • 缺乏自定义选项
  • 太过古老
  • 功能不够强大

什么是Windows 终端

Windows 终端首发于2019年2月，由微软公司研发，是一款面向命令行用户的新型、现代化、功能丰富、高效的终端应用程序。它包括Windows命令行社区最常要求的许多功能，包括对选项卡、富文本、全球化、可配置性、主题化和样式化等的支持。

Windows 终端有什么优点

  • 免费开源
  • 微软公司背书，GitHub社区高度参与，性能、支持有保障
  • 标签化，可拆分
  • 高度自定义
  • 可聚合各种命令行工具
  • 支持富文本

提醒

即使是现代化的 Windows 终端，调用的也是其它的命令行工具，而这些命令行工具默认都只能显示枯燥的文本。但是，有的命令行工具可以高度自定义，既可以产生丰富的视觉效果，又能提供丰富的信息，比如PowerShell，网络上已经有专门针对PowerShell进行美化的工具。

使用 Oh My Posh

了解Oh My Posh

Oh My Posh 是什么？

Oh My Posh是一个 Shell 提示符主题引擎。

我们已经安装了Windows 终端，可以使用选项卡、壁纸等设置，获得美观的外壳；使用PowerShell可以获得更加强大的命令行功能；但在很多特定场景下，默认的命令行界面能够显示的信息太少了，工作效率不高。
比如，在使用Git时，我们希望直观地看到当前所处的分支，以及更改、暂存、提交状态，我们来看看下面2张图的区别。

使用Windows 终端调用PowerShell，在普通状态下，几乎无法直观地获得任何信息。想要获得信息，就得输入各种繁琐的命令。再看看在Oh My Posh加持下的PowerShell效果：

效果不言而喻。使用了Oh My Posh，不仅可以显示当前 Git 分支名称，还能显示更改、暂存、提交的状态，甚至能识别出当前项目采用的是 Node.js 环境及其版本号18.16.1。这种将信息直接附加在提示符上的方式，省去了很多用命令查信息的时间。

Oh My Posh使用方法概述

要使用Oh My Posh，需要做四件事情：

  • 下载Oh My Posh可执行文件见3-3
  • 下载Oh My Posh主题文件夹见3-4
  • 创建并修改PowerShell配置文件见3-5
  • 下载、安装Nerd字体，在Windows 终端内设置相关字体见3-8

提示

可执行文件、主题文件夹、配置文件，建议放到同一个路径里，避免混乱。

因为PowerShell配置文件必须放在$HOME\Documents\PowerShell文件夹内，因此建议将Oh My Posh 可执行文件和themes文件夹也放在相同文件夹。

下载Oh My Posh可执行文件

访问 Oh My Posh 仓库发布页，选择需要的版本下载，得到的是一个可执行文件，将其存放到任意路径中，将该路径添加到环境变量。

提示

根据操作系统选择相应版本。比如：

  • Windows 64位版本：posh-windows-amd64.exe

下载主题文件夹

访问Oh My Posh 仓库：

下载Oh My Posh的项目压缩包。打开压缩包后，有一个themes文件夹，将其解压出来，也放到一个文件夹中最好是和Oh My Posh可执行文件放在一起。打开themes文件夹，你会发现里面都是.json和.yaml文件，每一个文件就是一个主题。

创建并修改PowerShell配置文件

关于PowerShell的配置文件，在官方文档中有详细的阐述。

配置文件有4种，从上往下优先级依次递增。也就是说，下方的配置文件，会覆盖上方的配置文件。因此，优先级最高的配置文件类型是：当前用户、当前主机。其中，Windows版本的路径有一个变量叫$HOME：

假设用户名为suyang，则$HOME代表C:\Users\suyang，也就是说创建当前用户，当前主机配置文件的实际路径应为：C:\Users\suyang\Documents\PowerShell，在此路径创建名为Microsoft.PowerShell_profile.ps1的文件：

用任意文本编辑器打开可能要用管理员权限打开，添加如下代码，并保存文件： 

      posh-windows-amd64 init pwsh --config 'C:/Users/suyang/Documents/PowerShell/Oh-My-Posh/themes/cloud-native-azure.omp.json' | Invoke-Expression

上述代码参考的是官方文档中的本地模式：

配置语句分为两种模式：

  • ①本地模式 初始化无需联网，但主题更新时，需要手动下载，覆盖旧文件。我用的是本地模式
  • ①远程模式 初始化需要联网，但会自动更新，无需人工干预。

提醒

  • 可执行文件名

    我的代码使用的文件名：posh-windows-amd64，
    官方文档使用的文件名：oh-my-posh；
    请根据自己下载的Oh My Posh可执行文件的实际文件名为准。

  • 管理员权限

    配置文件往往位于C盘下，自从Windows 10开始，对C盘的读写权限进行了更为严格的限定，部分软件包括记事本如果用普通模式打开C盘的文件，会无法正常保存，必须使用管理员权限打开这些软件。

    部分软件需要用管理员身份打开，才能顺利在C盘读写文件。

启动Windows 终端，点击顶栏的倒三角图标，选择PowerShell，大功告成！

更换主题

访问Oh My Posh官网主题预览页面，选择喜欢的主题。

点击喜欢的主题名称，进入该主题 GitHub 代码页，复制文件名，打开 PowerShell 配置文件，替换路径中的文件名，保存。此时重启 Windows 终端，或者在终端运行命令：. $profile，新主题即可生效。

使用Nerd字体

Oh My Posh主题中包含很多特殊符号，普通的字体包括绝大部分编程字体无法正确地显示这些符号，会导致难看的乱码。想要正常显示这些特殊符号，必须使用Nerd字体。

Nerd Fonts是一个专门下载 Nerd 字体的网站。

下载页面可以直接预览字体效果，选择你喜欢的字体，下载后安装。选用安装字体，请参考本地字体专题。

安装好字体后，重启终端，在设置里选择默认值→外观→字体，选择刚刚安装的字体，保存设置，即可享受 Oh My Posh 的最佳效果。
