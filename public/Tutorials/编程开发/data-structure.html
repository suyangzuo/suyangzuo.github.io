Data Structure & Algorithm

维基百科

在计算机科学中，数据结构是一种对数据进行组织、管理、存储的格式，目标是为了高效访问数据。更准确地说，数据结构是数据的集合、数据之间的关系、以及可应用于数据的函数和接口。
Tutorials Point

创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位。

Programiz

一个充满激情的小团队创建的学习资源网站，数据结构页面图文并茂，并且有“Python / Java / C / C++”4种语言的实现代码范例。

GeeksforGeeks

由印度人 Sandeep Jain 创办。 教程采用了模块化设计，除了传统章节外，还有小技巧、进阶教程等；每个数据结构都有多种语言的实现代码范例。

JavaTpoint

由印度人 Sonoo Jaiswal 创办。拥有极其详尽的数据结构章节、条目。

VisuAlgo

VisuAlgo最初由 Steven Halim 博士于2011年构想，旨在通过提供一个自定节奏的交互式学习平台，促进学生对数据结构和算法的更深入理解。
