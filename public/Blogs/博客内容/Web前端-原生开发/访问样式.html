访问内联样式

假设我们为一个id="盒子"的元素在CSS文件中编写了如下样式：

如果我希望在 JavaScript 中读取盒子的宽度，最直接的思维是如下写法： 

      
        const 盒子 = document.getElementById("盒子");
        const 盒子宽度 = 盒子.style.width; //读取盒子内联样式的宽度
        console.log(盒子宽度);

什么也没有！为什么？因为元素.style 读取的是 内联样式：

内联样式，也叫行内样式。我们有两种方法为元素添加内联样式：

  • 直接将内联样式写在HTML标签内： 
              
            <div id="盒子" style="width: 200px; height: 150px"></div>

    上述代码在div标签内部写入了一个style属性，这就是内联样式；而width和height又是这个内联样式的属性。有了内联样式，我们再次观察控制台：

    成功读取到内联样式中的宽度。

  • 在JavaScript代码中先设置内联样式，再读取： 
              
            const 盒子 = document.getElementById("盒子");
            盒子.style.width = "200px"; //先设置内联样式
            const 盒子宽度 = 盒子.style.width; //再读取内联样式
            console.log(盒子宽度);

    上述代码在第2行先设置了内联样式，我们可以在浏览器开发工具的元素选项卡中进行查看：

    盒子元素标签内出现了内联样式，这个内联样式并没有出现在我们的HTML代码内，而是由JavaScript代码直接添加到DOM树。现在观察控制台：

    成功读取到内联样式中的宽度。

访问计算样式

如果HTML标签内没有写内联样式，JavaScript也没有设置内联样式，怎么读取元素的样式呢？举个类似的例子：你想获取一扇门的宽高，最方便的方式是直接看设计图纸：宽1米，高2.5米，但假如没有设计图纸呢？每个人都知道答案：用尺去量。计算样式就是这样一种概念：找不到现成的样式，就自己算、自己量。 

      
        const 盒子 = document.getElementById("盒子");
        const 盒子宽度 = window.getComputedStyle(盒子).width; //读取盒子计算样式的宽度
        console.log(盒子宽度);

计算样式是只读的

计算样式只能读取，不能修改。因此你不能写这样代码： 

        
          const 盒子 = document.getElementById("盒子");
          window.getComputedStyle(盒子).width = "500px"; //错，计算样式无法修改

以上写法会在控制台显示如下错误：

错误信息提示： These styles are computed, and therefore the 'width' property is read-only ，翻译成中文：这些样式是计算出来的，所以 '宽度' 属性是只读的。
