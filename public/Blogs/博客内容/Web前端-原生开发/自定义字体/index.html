字体版权

很多人不重视字体的版权，甚至不知道字体是有版权的，这可能会导致版权方发来律师函甚至是直接索赔。在中国，最著名的案例就是 2007 年北大方正起诉暴雪在其由中国大陆代理运营的网络游戏《魔兽世界》中，未经授权使用了 5 款方正字体，并索赔的案件。相关信息可在国家版权局官网查到。因此，建立字体的版权意识是非常重要的。

字体类型

衬线字体Serif、无衬线字体Sans Serif

下图是衬线字体和无衬线字体的样本：

两种字体的区别很明显。衬线字体具有笔划末端的特征，无衬线字体没有这种特征。

  • 衬线字体：适用于古典、传统、有历史气息的场景。
  • 无衬线字体：适用于现代、科技、创造力的场景。永恒是一个专注于解决现代编程开发、设计的网站，因此基本上都是用无衬线字体

等宽字体Monospace、变宽字体

下面是用两种字体编写相同的代码，区别一目了然：

等宽字体：const element = document.querySelector("#username")
变宽字体：const element = document.querySelector("#username")
等宽字体：llllllllllWWWWWWWWWW
变宽字体：llllllllllWWWWWWWWWW

  • 等宽字体：所有字符宽度相同，适合编写代码。
  • 变宽字体：字符宽度有宽有窄，适合文章、书信。

使用授权字体

使用Google Fonts

为什么要用Google Fonts？

Google Fonts 官网FAQ 常见问题 页面很好地回答了这个问题：

谷歌字体都是免费的、可以用于商业用途，并且能在几乎所有媒介上使用，极大地避免了法律方面的问题。

访问Google Fonts官网，选择你喜欢的字体。如果希望筛选特定类型的字体，可以点击Filter过滤器，在左侧的过滤器中点击相应的选项：

选中Sans Serif，就会筛选出所有的无衬线字体。

选择字体后，点击右上角的Get Font，会将当前字体添加到字体选择库，并打开字体选择库界面，里面罗列出了所有你选中的字体：

此时点击Get embed code，就会弹出代码界面，直接给出了在使用该字体所需添加的代码：

我们是做 Web开发，因此选择Web选项卡。按照提示，将显示Embed code in the <head> of your html的代码，复制到head标签里；将显示CSS Class的代码，复制到CSS文件里，并对需要使用该字体的元素附上相同的类，就可以了。

删除Google 字体

随着你使用的字体越来越多，代码也会越来越冗长；这时你可能想删除一些用不到的字体：

打开字体选择库，点击删除按钮，即可删除不需要的字体，简化代码。
