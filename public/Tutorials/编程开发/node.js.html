瑞安·达尔
Ryan Dahl

详细生平

瑞安·达尔（1981年－至今）是美国的软件工程师，为可在编程语言 JavaScript 下运作的执行环境包“Node.js”、“Deno”的开发者。
瑞安·达尔在2009年推出名为“Node.js”的项目，他注意到 Flickr 网站在回应多个图片上传请求的处理上并不良好，想写一套能让网页服务器方便处理异步请求的软件框架，当时 Google 又推出了名为“V8”的 JavaScript 引擎，瑞安便采用 Javascript 编程语言在该引擎上编写出他所构想的内容。
Nodejs.org

Node.js 官网，有LTS版本和最新版本的详细API文档。
Nodejs.dev

使用 Gatsby.js 框架对 Nodejs.org 进行重新设计后的版本。
Tutorials Point

创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位。

W3 Schools

最知名的在线教程网站，每个月约6000万访问量。拥有多种语言的教程，其中以Web开发教程最为著名。

GeeksforGeeks

由印度人 Sandeep Jain 创办。教程分为多个模块，有详细的解释与代码示例。

JavaTpoint

由印度人 Sonoo Jaiswal 创办。除了基础教程外，还有使用Node.js访问数据库的教程等。

TutorialsTeacher

TutorialsTeacher 专为逐步学习Web技术而优化，其中的示例简单易懂，以提高易读性和理解能力。
