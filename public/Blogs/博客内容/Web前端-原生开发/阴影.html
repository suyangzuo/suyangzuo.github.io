盒子阴影

盒子阴影属性

访问交互式学习中心的盒子阴影页面，以更加直观、交互性的方式观察盒子阴影属性的影响。

盒子阴影有4个属性：

  • 水平偏移
  • 垂直偏移
  • 模糊
  • 扩展只有盒子阴影才有此属性 
      
        /* --------- 语法：--------- */
        box-shadow: 水平偏移 垂直偏移 模糊 扩展 颜色;
        
        /* --------- 范例：--------- */
        box-shadow: 5px 5px 10px 0 #84b507;

盒子阴影范例

概括：盒子阴影只受到盒子本身的影响，盒子里的内容对盒子阴影没有影响。

我是盒子

这个动画为我们展示了盒子阴影是如何受到盒子的影响的：盒子的圆角和缩放，会同步影响盒子阴影的形状和大小。

注意：动画中有一段盒子的背景消失了，这不是修改盒子的透明度文本依然正常显示，而是将盒子的背景色设置为透明。可以看出，背景色是否透明对于盒子阴影没有影响。

投影

投影属性

投影是滤镜的一种，访问交互式学习中心的滤镜页面，以更加直观、交互性的方式观察投影属性的影响。

投影有3个属性：

  • 水平偏移
  • 垂直偏移
  • 模糊
      
        /* --------- 语法：--------- */
        filter: drop-shadow(水平偏移 垂直偏移 模糊 颜色);
        
        /* --------- 范例：--------- */
        filter: drop-shadow(50px 25px 10px black);

投影范例

概括：投影只对盒子的可见部分有作用。

从动画中可以很直观地看出，盒子包括盒子的内容能看到什么，投影就是什么。

文本阴影

文本阴影属性

文本阴影有3个属性：

  • 水平偏移
  • 垂直偏移
  • 模糊
      
        /* --------- 语法：--------- */
        text-shadow: 水平偏移 垂直偏移 模糊 颜色;
        
        /* --------- 范例：--------- */
        text-shadow: 2px 2px 4px rgb(36, 78, 105);

文本阴影范例

概括：文本阴影只对文本有作用。

文本的影子

阴影叠加

阴影叠加语法

盒子阴影、投影、文本阴影，都可以多个阴影叠加在同一个元素上，语法也很简单： 

      
        // 盒子阴影叠加------------------------------------------------
        box-shadow:
        水平偏移 垂直偏移 模糊 扩展 颜色,
        水平偏移 垂直偏移 模糊 扩展 颜色,
        水平偏移 垂直偏移 模糊 扩展 颜色,
        ...........................
        水平偏移 垂直偏移 模糊 扩展 颜色;


        // 投影叠加---------------------------------------------------
        filter:
        drop-shadow(水平偏移 垂直偏移 模糊 颜色)
        drop-shadow(水平偏移 垂直偏移 模糊 颜色)
        ...................................
        drop-shadow(水平偏移 垂直偏移 模糊 颜色);


        //文本阴影----------------------------------------------------
        text-shadow:
        水平偏移 垂直偏移 模糊 颜色,
        水平偏移 垂直偏移 模糊 颜色,
        水平偏移 垂直偏移 模糊 颜色,
        .......................
        水平偏移 垂直偏移 模糊 颜色;
