长度单位说明
px 像素： 绝对长度单位，1px 等于屏幕上的一个像素点，是最常用的CSS长度单位，适合精确控制元素尺寸。
rem 根元素字体大小： 相对长度单位，相对于根元素 html 的字体大小。1rem = 根元素字体大小，适合响应式设计。
em 父元素字体大小： 相对长度单位，相对于父元素的字体大小。1em = 父元素字体大小，适合文本相关的尺寸设置。
vw 视口宽度： 相对长度单位，相对于视口宽度的百分比。1vw = 视口宽度的1%，适合响应式布局。
vh 视口高度： 相对长度单位，相对于视口高度的百分比。1vh = 视口高度的1%，适合全屏布局。
ch 字符宽度： 相对长度单位，等于数字 0 的宽度。适合文本相关的布局，如设置容器宽度为字符数的倍数。
