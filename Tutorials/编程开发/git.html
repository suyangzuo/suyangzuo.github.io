林纳斯·本纳第克特·托瓦兹
Linus Torvalds

详细生平

林纳斯·班奈狄克·托瓦兹，生于芬兰赫尔辛基市，拥有美国国籍，Linux内核的最早作者，随后发起了这个开源项目，担任Linux内核的首要架构师与项目协调者，是当今世界最著名的电脑程序员、黑客之一。他还发起了开源项目Git，并为主要的开发者。
Atlassian

Atlassian由2位大学校友 Mike Cannon-Brookes 和 Scott Farquhar 于2002年创建。其中包含了详细的Git教程。

git-scm

由Git社区成员维护的开源网站，也是Git的官方网站。拥有最详细的git指南。

Tutorials Point

创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位。

Simplilearn

Simplilearn是世界排名第一的在线训练营，也是世界领先的认证培训提供商之一。

Git How To

Git How To 可以引导你学完 Git 的基础； 俗话说得好：纸上得来终觉浅，绝知此事要躬行。
