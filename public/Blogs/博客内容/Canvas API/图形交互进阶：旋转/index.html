旋转操作热区模式

环绕式

这应该是我们最熟悉的一种热区模式，热区环绕在图形的四周，但是又和图形有一定的间距。

Photoshop使用的就是这种热区模式。

四角式

这种热区模式也比较流行，在图形的四个角放置热区，代表旋转操作。

Inkscape使用的就是这种热区模式。

全覆盖式

这种热区模式相对冷门一点，首先要让当前的操作模式变成旋转绘图软件一般用选中旋转工具的方式实现，整个绘图区域都是热区，在任何地方拖拽都可以实现旋转操作。

GIMP使用的就是这种热区模式。
