制作Windows系统引导盘原创

作者： 苏扬

日期： 2024年2月26日

历史与现状

引导盘工具的历史

自从Windows XP时代开始，定制化的系统引导盘制作工具大量出现，最有名的莫过于番茄花园。系统的批量盗版，可以追溯到1995年，这一年，Windows 95正式发售；也是这一年，光盘和光驱开始流行起来。到了 21 世纪的起始 10 年，Windows XP大获成功，盗版系统也大行其道，很多盗版商在原始操作系统里加入了定制选项以及预装软件，将其封装成一整套预装系统，这样一来，用户只要安装好系统，很多必备软件也同时装好，省去了很多麻烦，毕竟在Windows系统上安装软件是个很麻烦的事情。正因如此，这种盗版预装系统曾经占据了非常大的市场。番茄花园只是其中一个，其它类似的萝卜家园、大白菜、U大师、U深度、雨林木风、老毛桃等等等等……，数不胜数。

盗版本身还有盗版，你甚至分不清哪个才是真正的大白菜。

然而，这样的系统也有很大的弊病。最惹人讨厌的就是其中的广告和流氓软件恶意软件；而最大的问题则是这些工具内置的系统均为盗版。最有名的案例就是2008年 商业软件联盟 诉 番茄花园 侵犯著作权案，该案件在微软官网也有记载。

总之，这种预装系统，优点是非常方便，就像拎包入住一样；缺点则是广告、恶意软件、盗版。

引导盘工具的现状

现今的Windows系统引导工具，盗版情况依旧存在，但比起十几年前已有很大的改观。新手用户更多地用的是OEM 预装系统，均为正版。而专业用户喜欢自己安装，但绝大多数都倾向于安装原版系统，也就是没有经过任何第三方更改的原始系统。

小知识

原版系统指的是微软发布的原始光盘镜像扩展名为 .iso。这种镜像文件本身没有正版或盗版的概念。当你用这个镜像安装好系统后，系统处于未激活状态，如何激活系统，决定了你用的是正版还是盗版。

如果你是花钱购买了序列号或者其它正规渠道激活的，那就是正版；如果你是通过激活软件或者其它非正规渠道激活的，那就是盗版。

可以在Windows 11的设置中查看激活状态。

与此同时，出现了很多优秀的引导工具，它们无一例外，都不会在原版系统里预装软件可能会提供一些修改特定设置的选项，而这些选项是有用的，而是为用户提供更加优秀的引导体验，这些工具界面清爽，功能简约而强大。而本博客的主旨，就是为大家推荐优秀易用的引导工具，当然，它们都是免费开源的 。

准备工作

U盘

基本原则是：U盘的容量，要大于系统镜像的容量，现在的Windows系统的镜像比以前大很多。

Windows 11 23H2 64位版本的镜像文件已经高达 6.16GB，因此U盘至少也要 8GB 的容量。

特别注意：制作启动盘，会删除 U 盘所有数据！制作之前请备份好 U盘 的重要数据！

系统镜像文件

微软官网提供了Windows 原版镜像的 下载页面，选择你想要的 Windows，进入下载分页：

此分页中其它都不重要，我们只需要光盘镜像，选择想要的版本，点击下载即可。

同代Windows的不同版本

每一代 Windows 都根据功能划分为不同的版本，也对应不同的价格。大致上有 4 个版本：

  • 家庭版
  • 专业版
  • 企业版
  • 教育版

家庭版的功能最弱，强烈建议不要选。

multi-edition的意思是多版本，这个镜像同时包含了专业版、企业版、教育版，没有特殊原因，我们都会选择多版本镜像可能还会多出一个专业工作站版本。

具体选择哪个，我个人认为，只要不是家庭版，都可以。

查看TPM 2.0支持情况

从Windows 11开始，要求主板必须具备TPM 2.0可信平台模块 2.0方可安装。如何得知自己电脑的主板是否拥有TPM 2.0模块呢？

下载HWiNFO Windows PortablePortable：便携版，无需安装：

依次展开主板→ACPI设备，如果能看到Trusted Platform Module 2.0每个单词首字母组合即为 TPM 2.0，就表示你的主板拥有TPM 2.0模块，可以正常安装Windows 11。

Ventory推荐

Ventory简介

  • 官方网站：https://www.ventoy.net/cn/
  • GitHub：https://github.com/ventoy/Ventoy

Ventory到底有什么优势？为什么要推荐它？
传统的引导盘制作工具，每次制作都需要格式化U盘，导致U盘内的数据全部丢失。同时，因为需要把光盘镜像包含的文件全部写入U盘，整个制作过程也比较漫长，且每个U盘只能引导 1 种操作系统。
Ventory则无需写入镜像内的文件，只要拷贝镜像本身就行，因此制作时间大大缩短，并且可以拷入多个镜像，实现多个系统选择。同时，因为只需要拷贝镜像文件就可以添加系统，因此无需反复格式化U盘，非常地方便。Ventory 在 GitHub 上甚至获得了 56000 多 ！
本人现在重装系统全部用Ventory。

Ventory使用指南

双击Ventoy2Disk.exe，打开Ventory。首先是选择U盘：

注意不要选错。选好U盘后，有几个重要的配置选项需要更改：

安全启动对应 BIOS 里的Secure Boot，建议勾选。

选择分区类型。如果主板支持GPT就用GPT；不支持则用MBR只有非常古老的主板才会不支持 GPT。

点击分区设置，弹出分区设置对话框：

有多种文件系统可选，根据实际需求选择，其它选项不用改：

  • exFAT：用于 U盘，可以写入大于 4GB 的单文件。
  • NTFS：用于硬盘，比如移动固态硬盘。
  • FAT32：无法写入大于 4G 的单文件，但是很多 BIOS 只认 FAT32，刷 BIOS 用这个。
  • UDF：用于光盘。

以上配置选项改好，就可以开始制作启动盘了：

如果 U盘 是第一次制作成启动盘，就点击安装，此时会弹出警告：

初次制作，需要格式化 U盘，这个操作会清除 U盘 内所有数据Ventory的优点之一就是只需要格式化 1 次。点击是：

会再次让你确认是否格式化，确认无误就点击是，开始制作：

Ventory会显示制作进度，耐心等待：

制作成功后，会弹出提醒对话框，点击确定。

制作成功后，Ventory的界面有一些变化，设备内部也就是 U盘 内部也拥有了 Ventory；以后如果 Ventory 版本更新，同一个 U盘 只要点击升级即可，升级不用格式化U盘。
此时，你会发现电脑上多出一个 U盘 分区，这个分区就是专门给你复制各种系统镜像文件的当然也可以复制其它各种数据资料，这就是Ventory的一大优点：镜像文件无需展开，直接复制镜像即可。这也为 Ventory 进行多个系统引导奠定了基础。

Ventory另一个优点就体现在此，引导分区和数据分区是互相独立的。当你升级 Ventory时，影响的是引导分区平时不可见，而各种个人资料、数据、系统镜像，则是复制到数据分区。

点击启动：，会在浏览器打开插件页面：

插件页面的默认起始页是设备信息，左边的菜单提供众多的插件功能，但我们最需要的其实就 2 个，它们都在全局控制插件页面内。

Windows 11 要求主板必须具备TPM模块，否则无法安装。另外 Windows 11 还会检查你的电脑是否具备 4GB 以上的内存，是否支持安全启动。想绕过此检查，选择1即可。

在线账户需求也是个麻烦的事情。如果你有类似的账户比如 微软账户，那没问题，直接登录。但很多人没有注册在线账户，或者不想使用在线账户，甚至根本没有网络，那干脆把这个过程屏蔽掉，选择1即可。

注意：插件设置完成后，需要重新制作或升级Ventory 启动盘方可生效。

Rufus

Rufus简介

  • 官方网站：https://rufus.ie/zh/
  • GitHub：https://github.com/pbatard/rufus

Rufus属于传统型引导盘制作工具，也就是说，每次制作都需要格式化U盘，且只能引导 1 种操作系统。类似的工具有很多，但Rufus是其中的佼佼者：界面清爽、免费开源、体积很小仅有 1.4MB，且整个程序仅有 1 个文件，便于传输携带。在 GitHub 上拥有 26000 多 的收藏量。

Rufus使用指南

双击Rufus可执行文件，打开 Rufus：

在设备中选择自己的 U盘。

首先点击选择按钮，选择系统镜像文件。

找到系统镜像文件，选择并打开。

选择镜像后，引导类型选择下面会出现镜像文件名，并且右边的变成蓝色。

Windows To Go的意思是把系统直接安装到U盘上。绝大多数情况下，我们都希望把系统安装到硬盘里，因此应该选择标准 Windows 安装。

和 Ventory 类似，一般都选GPT；非常古老的电脑，选MBR。

正常情况都选UEFI。实际上，UEFI和GPT是配套使用的。

文件系统只有NTFS可选。因为 Rufus 不区分引导分区和数据分区，二者是合并在一起的，而exFAT无法引导，所以只能用NTFS，这就是 Rufus 不如 Ventory 的地方。

簇大小不用改，默认即可。

当状态为准备就绪时，点击开始按钮。注意：当鼠标悬停在开始按钮上时，会弹出提示，说明此操作会删除 U盘 上所有数据。

点击开始后，会弹出Windows 用户体验对话框，前 2 项的功能，和在Ventory 插件一节中提到的功能类似。后面 4 项则是系统安装好之后，正常情况下会经历的步骤，如果想提前设置好，可以自行设置。设置完毕后，点击OK，即可开始制作过程，耐心等待制作完毕。
