自定义滚动条原创

作者： 苏扬

日期： 2024年6月17日

滚动条元素

为什么要自定义？

一切都是为了用户体验，可能是默认的滚动条太细，选起来不方便；也可能是滚动条的样式和网站的整体色调不一致，感觉很突兀，比如：

网站整体色调接近黑色，滚动栏却是白色，显得非常刺眼；因此，我们经常需要定制滚动条的样式，以符合网站的整体风格。

滚动条元素

滚动条在DOM树中实际上是伪元素，分为几个部分：

  • ::-webkit-scrollbar：滚动条本身。如忽略，则其它部分设置无效
  • ::-webkit-scrollbar-track：滚动条轨道
  • ::-webkit-scrollbar-thumb：滚动条滑块
  • ::-webkit-scrollbar-button：滚动条两端的箭头

提醒

webkit代表的是源于webkit核心打造的浏览器，主要代表有Chrome、Edge、Safari，因此这些伪元素对于FireFox浏览器是无效的。

其次，这些伪元素是非标准的，MDN Web Docs 相关页面明确提出尽量不要在生产环境中使用。简单地说就是公司的正规项目里不要用这些伪元素。

尽量不要在正式项目里使用这些伪元素。

自定义滚动条样式

滚动条

滚动条可以修改宽度和背景：

      
        body::-webkit-scrollbar {
          width: 35px;
          background-color: darkgoldenrod;
        }

滚动条的宽度和背景已改变。

提醒

::-webkit-scrollbar的宽度和背景色至少要写一个，其它的伪元素才能起作用。

滚动条的背景不仅能用颜色，也能用渐变：

      
        body::-webkit-scrollbar {
          width: 25px;
          background-image: linear-gradient(seagreen 10%, lightskyblue 90%);
        }

滚动条轨道

滚动条轨道主要是修改背景，注意：滚动条轨道的背景会覆盖滚动条的背景： 

      
        body::-webkit-scrollbar {
          width: 25px;
          background-image: linear-gradient(seagreen 10%, lightskyblue 90%); /* 被第7行覆盖 */
        }
        
        body::-webkit-scrollbar-track {
          background-color: rgb(170, 87, 115); /* 此样式会覆盖第3行 */
        }

::-webkit-scrollbar-track的背景会覆盖::-webkit-scrollbar的背景。

滚动条滑块

现在在滚动条上还看不到滑块，不是没了，而是修改了::-webkit-scrollbar后，滑块会自动变成透明。因此我们需要为滑块手动添加样式代码： 

      
        /* 滚动条 */
        body::-webkit-scrollbar {
          width: 25px;
        }

        /* 轨道 */
        body::-webkit-scrollbar-track {
          background-color: rgb(71, 71, 71);
        }

        /* 滑块 */
        body::-webkit-scrollbar-thumb {
          background-color: rgb(58, 151, 115);
        }

修改背景之后就能看到滑块了。我们还可以进一步定制其样式： 

      
        body::-webkit-scrollbar-thumb {
          background-color: rgb(58, 151, 115);
          border-radius: 999px; /* 边框圆角 */
        }

两端改成圆角也没有问题。再为滑块的:hover和:active伪类添加不同的样式： 

      
        body::-webkit-scrollbar-thumb {
          background-color: seagreen;
          border-radius: 999px;
        }

        body::-webkit-scrollbar-thumb:hover {
          background-color: rgb(96, 207, 166);
        }

        /* :active 表示鼠标按住滑块不放 */
        body::-webkit-scrollbar-thumb:active {
          background-color: rgb(159, 207, 249);
        }

这样一个自定义的滚动条就算是基本完成了。
