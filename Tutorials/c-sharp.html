安德斯·海尔斯伯格
Anders Hejlsberg

详细生平

安德斯·海尔斯伯格（Anders Hejlsberg，1960年12月－），丹麦人，Borland Turbo Pascal编译器的主要作者。进入微软公司后，先后主持开发了 Visual J++、.Net、C# 和 TypeScript。
C#文档

C#由微软开发，微软官方的C#文档可称第一手资料，内容之详细举世公认。不过文档中的叙述有些“官方化”，对刚刚接触编程的人来说不是很容易理解。

.NET API 浏览器

同样是微软官方.NET文档的一部分，可以说是.NET API 查询的最佳途径，列举了所有API的使用方法。查询时记得选择适合的.NET版本。

C# Corner

公司总部位于宾夕法尼亚州，是一个专注于C#的开发人员在线社区，提供分享C#的新闻、文章、教程等服务， 并在社区论坛中进行讨论，同时还提供职业发展工具，如职业建议、简历撰写、培训、认证、书籍和白皮书以及视频。

Tutorials Teacher

TutorialsTeacher 专为循序渐进学习Web技术而优化，范例都很简洁以提高可读性和理解程度。站内包含了多种技术栈的教程。

极客教程

包含了大量技术栈的教程，以及一些技术类文章。
C语言中文网

成立于2012年初，由站长 “严长生”在大三寒假期间创办。虽然名称叫“C语言中文网”，但也包含了大量其它技术栈的教程。

Tutorials Point

创建于2006年的在线教程网站，文本库和资源均免费提供。教程条目清晰，案例详实，对各种概念解释得也很到位。

Javapoint

由印度人 Sonoo Jaiswal 创办。网站提供了大量技术栈的教程，甚至连怎么用 Gmail 都有……

GeeksforGeeks

由印度人 Sandeep Jain 创办。
网站提供免费教程、数百万篇文章、现场、在线和课堂课程、频繁的编码竞赛、行业专家的网络研讨会等。 教程内容非常详尽、清晰。

WPF Tutorial

专门讲解WPF框架的网站，有简体中文，建议具备一定的C#基础后再看。
