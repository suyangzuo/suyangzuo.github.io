详细生平

詹姆斯·高斯林（1955年5月19日－），加拿大软件专家，Java编程语言的共同创始人之一，一般公认他为“Java之父”。
2004年，詹姆斯·高斯林因Java编程语言架构的构思和开发以及对Windows系统的贡献而被选为国家工程学院成员。
Oracle Java SE Docs

Oracle官方的Java文档，包含了所有关于Java的知识库以及API解释。

W3 Schools

W3 Schools 是全球最为知名的教程网站，以Web教程为主，也包含其它教程，如Java。

JavaTpoint

由印度人 Sonoo Jaiswal 创办。其中的Java教程不仅包含基础章节，还加入了进阶主题。

Tutorials Point

创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位，包含基础和进阶内容。

Programiz

一个充满激情的小团队创建的学习资源网站。Java教程被分为多个模块，每个模块包含多个子章节，图文并茂，配有相应的范例代码。

GeeksforGeeks

由印度人 Sandeep Jain 创办。
教程采用了模块化设计，除了传统章节外，还有小技巧、进阶教程等。每个章节都配有 Java 代码演示。

Guru99

Guru99是由 Krishna 创办的在线教程网站，宗旨是致力于让大众负担得起优质教育。提供了详实的Java教程。

Scaler

由一群印度人创办。Scaler是为工作技术专业人士提供的在线变革性技能提升平台。对年轻专业人员的教学和培训采取了经过行业审查的方法。

BeginnersBook

BeginnersBook 是一个针对初学者的教程网站，包含了多种技术栈的主题。

Jenkov.com

由 Jakob Jenkov 创建的技术与媒体网站，包含了很多教程与技术文章。
