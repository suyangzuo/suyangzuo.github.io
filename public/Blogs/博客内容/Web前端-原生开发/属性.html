属性原创

作者： 苏扬

日期： 2023年6月19日

3种语言，3种属性

HTML属性 Attribute

假设在HTML文件内有如下元素： 

      
        <a class="联系方式链接" href="https://www.artstation.com/" target="_blank">
        欢迎访问 ArtStation 
      </a>
        
      <!-- 如果属性和值写在一行内，代码过长；可将属性分行书写，效果相同： -->

      <a 
        class="联系方式链接" 
        href="https://www.artstation.com/"
        target="_blank">
        欢迎访问 ArtStation
      </a>

这是一个a标签，表示超链接元素。在这个元素中，有3个HTML 属性：

  • class，属性值为：联系方式链接
  • href，属性值为：https://www.artstation.com/
  • target，属性值为：_blank

CSS属性 Property

假设一个CSS选择器包含了如下样式： 

      body {
        width: 100%;
        min-height: 100vh;
        background-color: #354045;
        color: white;
      }

这是一个body元素选择器，其中包含4个CSS 属性：

  • width，属性值为：100%
  • min-height，属性值为：100vh
  • background-color，属性值为：#354045
  • color，属性值为：white

提醒

有时，我们会把CSS样式写在HTML标签内部，称为：行内样式或内联样式： 

        <div 
          id="交互容器" 
          class="容器" 
          style="width: 100%; height: 100%">
        </div>

此时，id、class、style是 3 个HTML 属性，而width和height则是 2 个CSS 属性。

JavaScript属性 Property

假设有这样一段JavaScript代码： 

      const student_001 = {
        name: "张三",
        gender: "男",
        age: 21,
        grade: "2024",
        classNumber: "2465",
      };

这是一个JavaScript对象，定义了5个JavaScript 属性：

  • name，属性值："张三"
  • gender，属性值："男"
  • age，属性值：21
  • grade，属性值："2024"
  • classNumber，属性值："2465"
