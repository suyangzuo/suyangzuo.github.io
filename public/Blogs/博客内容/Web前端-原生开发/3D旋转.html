首先，直观地观察一下2D和3D旋转的区别：

虽然都是旋转，但3D旋转展现出了明显的立体效果，更符合现实世界中的真实维度。

透视

现实世界中的物体本来就是有近有远。然而在电脑屏幕上，所有物体与你的眼睛的距离都是一样的，这意味着所有的物体都处于同一平面。怎样在平面上展现三维效果呢？

学过美术的人肯定知道答案：透视。透视的作用就是在 2D平面上展现3D立体效果。

把美术的概念挪用到代码上，就要在元素样式中加上透视perspective属性。

3D旋转

添加透视

为了获得3D效果，我们需要为元素添加透视，这里有两个注意点：

  • 添加透视的元素，必须永远与视线垂直，因此该元素不能旋转。
  • 透视的距离，既不能太远，也不能太近。太远会看不出3D效果，太近则透视效果过于夸张。 
        
          .透视元素 {
            perspective: 500px; /* 此距离不能太近，也不能太远。具体数值需要自行测试 */
          }

添加变换风格

透视元素不能旋转，那需要旋转的元素，就要作为透视元素的子元素。旋转是变换transform的一种，而变换风格默认是平面，我们需要将其改成3D。 

        
          //透视元素在外，变换元素在内
          <div class="透视元素">
            <div class="变换元素"></div>
          </div>
        
      
        
          .变换元素 {
            transform-style: preserve-3d; /* 此属性默认值为 flat */
          }

添加旋转属性

现在，只要为变换元素添加旋转属性，即可看到3D旋转效果： 

        
          .变换元素 {
            transform-style: preserve-3d; /* 此属性默认值为 flat */
            transition: 1s; /* 过渡，否则旋转会一瞬间完成，看不出效果 */
          }
          
          .变换元素:hover {
            //以下两种写法效果一致
            rotate: y 180deg; /* 较新的写法 */
            transform: rotateY(180deg); /* 传统写法 */
          }

透视距离设置为500px，3D效果比较好。

透视距离设置为100px，3D效果非常夸张，现实中类似于贴脸观察。

透视距离设置为5000px，几乎感受不到任何3D效果。

卡片翻转

3D旋转的应用

旋转的一个常见应用场景，就是做翻转的卡片。不过这里有一个问题：我们希望卡片翻转过来后是空白的，而不是像之前一样只是把原有的内容翻转过来。

因此，我们需要在旋转元素的内部，放置两个元素。一个元素放入正面内容，一个元素放入反面内容。当旋转元素开始旋转时，正面和反面两个内部元素也会随之旋转。 

        
          //透视元素在外，变换元素在内
          <div class="透视元素">
            <div class="变换元素">
              <div class="正面元素">
              <div class="反面元素">
            </div>
          </div>

为了能更好地观察正面和反面元素，我先将它们分开，并且隐藏了透视元素和变换元素。

隐藏反面

相信你已经看出问题所在，卡片翻转过来后，不是空白。这里需要使用CSS属性：反面可见性。 

        
          .正面元素,
          .反面元素 {
            backface-visibility: hidden; /* 反面-可见性：隐藏 */
          }

好多了，不是吗？不过，两张图片应该是一正一反，因此，反面元素平时就应该水平翻转180°： 

        
          .反面元素 {
            rotate: y 180deg;
          }
          
          .正面元素,
          .反面元素 {
            backface-visibility: hidden; /* 反面-可见性：隐藏 */
          }
