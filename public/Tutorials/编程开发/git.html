林纳斯·本纳第克特·托瓦兹
Linus Torvalds

详细生平

林纳斯·班奈狄克·托瓦兹，生于芬兰赫尔辛基市，拥有美国国籍，Linux内核的最早作者，随后发起了这个开源项目，担任Linux内核的首要架构师与项目协调者，是当今世界最著名的电脑程序员、黑客之一。他还发起了开源项目 Git，是该项目的开发者。
Atlassian

Atlassian 由2位大学校友 Mike Cannon-Brookes 和 Scott Farquhar 于2002年创建。其中包含了详细的Git教程。

git-scm

由 Git 社区成员维护的开源网站，也是 Git 的官方网站。拥有最详细的 Git 指南。

Tutorials Point

创建于2006年的在线教程网站，条目清晰，案例详实，对各种概念解释得也很到位。

Simplilearn

Simplilearn 是世界排名第一的在线训练营，也是世界领先的认证培训提供商之一。

Git How To

Git How To 可以引导你学完 Git 的基础； 陆游在800多年前就说过：纸上得来终觉浅，绝知此事要躬行。

The Pragmatic Git

由瑞典人 Alexis Määttä Vinkler 创建的博客，目的是帮助开发者提升 Git 专业使用技能。

Initial Commit

由美国人 Jacob Stopak 创办的技术网站，主要内容是 Git 的使用，但也涵盖了其它开发主题。
