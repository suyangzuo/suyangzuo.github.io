详细生平

尤雨溪是出生于中国江苏省无锡市的程序员，是 Web应用框架 Vue.js 的发起者和开发者。2005年尤雨溪前往美国念书，就读于科尔盖特大学及帕森斯设计学院，后任职于 Google。2016年，尤雨溪成立 Vue Technology LLC 公司。
Vue School

通过高级教程和视频课程，从核心团队成员和行业专家那里学习 Vue.js 和现代前沿前端技术。

Vue Mastery

作为 Vue.js 开发人员的终极资源，Vue Mastery 每周都会提供课程，这样你就可以学习成为 Vue.js 开发者所需要的东西。
